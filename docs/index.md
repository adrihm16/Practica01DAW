# Mi Programa

## Descripción

Esta es una calculadora simple en Java que permite realizar operaciones básicas y algunas avanzadas.  
Incluye suma, resta, multiplicación, división, cálculo de potencias y raíces cuadradas, todo con documentación generada automáticamente.

## Características

- ✅ Sumar dos números (`sumar()`)
- ✅ Restar dos números (`restar()`)
- ✅ Multiplicar dos números (`multiplicar()`)
- ✅ Dividir dos números (`dividir()`)
- ✅ Calcular la potencia de un número (`potencia()`)
- ✅ Calcular la raíz cuadrada de un número (`raizCuadrada()`)

## Tecnologías

- Java 11+
- Documentación generada con Javadoc
- Sitio creado con MkDocs Material
